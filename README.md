## Sublime Text 3

```bash
sudo add-apt-repository ppa:webupd8team/sublime-text-3
sudo apt-get update
sudo apt-get install sublime-text-installer
```

## Fix Ubuntu script

```bash

```

## Remove Unity scrollbars

```bash
gsettings set com.canonical.desktop.interface scrollbar-mode normal
```
